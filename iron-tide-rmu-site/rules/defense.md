# Defense Rules Reference

## Defensive Bonus (DB) Components
DB = Quickness DB + Shield/Dodge + Parry + Cover + Situational

### Quickness DB
- Quickness stat bonus  3
- Always active as long as combatant can move and is not surprised/flatfooted
- Negative Qu bonus = negative DB (easier to hit than average)

### Passive Dodge
- Add Running ranks to DB (up to +50 max)
- No AP cost  always active
- **Cannot combine with Passive Block (Shield)**
- Halved vs ranged attacks
- Armor and encumbrance penalties apply directly to dodge DB
- Injury penalties reduce dodge DB

### Partial Dodge
- Requires concentration
- Add half of Movement skill bonus to DB

### Full Dodge
- Costs 4 AP
- Add full Movement skill bonus to DB

### Passive Block (Shield)
- Add Shield skill ranks to DB (up to +50 total including shield base DB)
- **Cannot combine with Passive Dodge**
- Shield base DB varies by type:
  - Target Shield: +15 (1 attack)
  - Normal Shield: +20 (2 attacks)
  - Full Shield: +25 (3 attacks)
  - Wall Shield: +30 (4 attacks)
- Shield defends against listed number of attacks; shield maneuver required for each beyond the first
- **Using shield to attack = lose ALL shield DB for the round**

### Full Block (Shield)
- Costs 4 AP
- Add full Shield skill bonus to DB

### Parrying
- Allocate OB to DB against one foe
- Reduces attack OB by same amount
- Can parry multiple foes: Absolute maneuver roll (using Combat Training skill) required for each foe beyond the first
- If all OB spent on parry, no attack this round (but Riposte talent can trigger)
- Parrying from flank/rear: penalized by positional modifier, offset by Reverse Strike skill

### Restricted Parrying (2 OB = 1 DB)
- Unarmed vs armed opponent (unless wearing bracers/gauntlets)
- Fighting invisible foe
- Fighting incorporeal foe

### Cover (DB bonus)

| Cover | Melee | Ranged |
|-------|-------|--------|
| Partial | +10 | +20 |
| Half | +20 | +40 |
| Full | +50 | +100 |

Hard cover (walls, trees) uses listed values. Soft cover (fog, darkness) uses same values.
