# Combat Resolution Rules Reference

## Attack Roll
- Roll: **d100 open-ended** + OB - DB
- Open-ended: 96+ roll again and add; 01-05 roll again and subtract
- Look up result on weapon's attack table vs target's Armor Type (AT)
- Result gives: hits dealt + critical type/severity (if any), e.g. "12BP" = 12 hits + B Puncture critical
- Attack total over 175 = Absolute Hit: +1 to critical roll per 5 above 175 (round up)

## Spellcasting Roll (SCR)
- Roll: **d100 open-ended** + List Ranks + Realm Stat Bonus + List Type + Other Modifiers
- List Type: +5 own Base, 0 Open, -5 Closed, -10 other
- **Succeeds on result of 1 or greater**
- Result of 0 = spell fizzles, PP lost, no failure roll
- Negative result = roll on Spell Failure table
- The SCR result becomes the number enemies must beat on Resistance Rolls
- Overcasting penalty: -20 per spell level above caster level
- Hasty casting: -25 per AP reduced below 4
- Preparation: +10 per round of preparation (max +20)

## Critical Roll
- Roll: **Flat d100. NOT open-ended.**
- Usually NO modifiers
- Exceptions that DO modify the critical roll:
  - Ambush skill (adds based on ranks when attacking from flank/rear)
  - Slaying attacks
  - Absolute Hit (+1 per 5 above 175 on the attack roll)
- Critical table rows go 1-100 maximum
- Critical severity ranges A (minor) to E (devastating)
- Severity above E: E critical + secondary critical per Table 9-8a (F=E+A, G=E+B, H=E+C, etc.)

## Resistance Roll (RR)
- Roll: **d100 open-ended** + stat bonus + racial RR + level bonus (+2/level) + realm bonus (+10 own realm)
- Must meet or exceed the caster's SCR result to resist
- Stat by type: Physical=Co, Fear=SD, Channeling=In, Essence=Em, Mentalism=Pr

## Fumbles
- If **unmodified** attack roll  weapon's fumble number, roll on fumble table immediately
- Fumble number reduced by 1 per 5 ranks in the attack skill (minimum 1)
- Fumble is checked BEFORE modifiers are applied

## Stunned Foe Bonus
- +20 OB when attacking a stunned foe

## Surprised/Flatfooted Foe
- +25 OB bonus
- If in melee, also treated as rear attack (+35 OB)
- Surprised foe gets no Quickness DB

## Flanking Modifiers
- Flank attack: +15 OB
- Rear attack: +35 OB
- Moving to flank: -30 OB penalty
- Moving to rear: -70 OB penalty
- Parrying from flank/rear: penalized by positional modifier, offset by Reverse Strike skill
