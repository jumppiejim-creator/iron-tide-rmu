# Size and Positional Rules Reference

## Size Categories
| Size | Category | Hit Multiplier | Examples |
|------|----------|---------------|----------|
| I | Tiny | 0.1 | Insects |
| II | Miniscule | 0.25 | Rats |
| III | Small | 0.5 | Halflings, Goblins |
| IV | Small | 0.75 | Dwarves |
| V | Medium | 1.0 | Humans, Elves, Gnomes |
| VI | Big | 1.5 | Trolls |
| VII | Large | 2.0 | Ogres, Giants |
| VIII+ | Huge+ | 2.5+ | Dragons |

## Size Adjustments to Attacks
- Attacker larger than target: +1 critical severity per size difference; target gets +5 DB per size smaller
- Attacker smaller than target: -1 critical severity per size difference
- Example: Big (VI) Grond attacking Medium (V) target = +1 crit severity, target gets +5 DB
- Example: Medium (V) smuggler attacking Big (VI) Grond = -1 crit severity

## Size and Charging
- If charging at Jog pace or faster AND attacker is larger than target: additional +1 size to attack

## Restricted Quarters
Penalties when fighting in spaces too small for your character:

| Restriction | Penalty |
|-------------|---------|
| Close | -25 |
| Cramped | -50 |
| Tight | -75 |
| Confined | -100 |

- Restricted Quarters skill offsets these penalties
- Grond (9' tall) will face these penalties in most indoor environments
- Grond's Restricted Quarters skill: +30 (fully offsets Close, reduces Cramped to -20)

## Melee Range
- Melee range = (height/2) + weapon length
- Grond: 4.5' + weapon length (~3' warhammer) = ~7.5' reach
- Medium human: ~3' + weapon length

## Flanking and Positional
| Position | OB Modifier |
|----------|-------------|
| Frontal | +0 |
| Flank | +15 |
| Rear | +35 |
| Moving to Flank | -30 |
| Moving to Rear | -70 |

- Parrying attacks from flank/rear is penalized by the positional modifier
- Reverse Strike skill offsets flanking/rear parry penalties
- Ambush skill adds to critical roll when attacking from flank or rear
