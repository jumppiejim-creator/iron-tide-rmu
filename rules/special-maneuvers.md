# Special Combat Maneuvers Reference

## Called Shots

Sometimes a combatant may wish to attack a specific location on the body (head, arms, chest, abdomen, legs).

### Penalty
- Standard penalty: **-25 to attack roll** against same-sized humanoids
- GM may adjust from -10 to -50 based on circumstances
- Example: Human attacking a giant's leg with dagger = -10; attacking giant's head = -50

### Armor on Called Location
If the targeted area is armored differently from the torso:
- No armor: AT 1
- Light armor: AT 3
- Medium armor: AT 5
- Heavy armor: AT 9

### Resolving Called Shot Criticals
- If attack succeeds, the hit is on the specified location
- If a critical is rolled and the result indicates a DIFFERENT location, keep checking lesser results on that column until you find one affecting the called body area
- If no lesser row matches, use the 01 result and modify the description to fit
- The special Groin/Abdomen result on row 66 cannot be obtained by this shift; use the 26-35 result instead

## Disarming

A disarming attack forces an opponent to drop their weapon.

### Resolution
- Resolved as a conflicting maneuver
- Attacker rolls: normal attack roll (including target's parry penalty) + Disarming skill - table penalties
- Defender rolls: attack roll (subtracting any parry used) as defense
  - Defender roll over 100: excess penalizes attacker
  - Defender roll under 100: shortfall gives bonus to attacker

### Disarm Penalties

| Attack Type | Modifier |
|-------------|----------|
| Unarmed/grapple/crush | -50 |
| Blade attack | -75 |
| Piercing attack | -100 |
| Ranged attack* | -100 |
| Disarming a 2H weapon* | -25 |

*Cumulative with other penalties

- Combat Expertise: Disarming skill offsets these penalties (to minimum 0)
- Cannot disarm natural attacks (claws, bite, etc.)

### Results

| Roll | Result |
|------|--------|
| < 1 | Absolute Failure: You fumble your own weapon |
| 1-75 | Failure |
| UM 66 | Unusual Event: If you fail, foe disarmed YOU. If succeed, both weapons drop. |
| 76-100 | Partial Success: Treat as normal attack instead |
| 101-175 | Success: Foe's weapon falls to ground within melee range |
| 176+ | Absolute Success: Take the weapon or throw it up to 10' |

## Firing into Melee

- Defender may gain cover from other combatants based on positioning
- If ranged attack misses intended target, it may hit a combatant providing cover
- Second attack roll at +0 skill OB against secondary target (randomly selected)
- Secondary target does not benefit from cover modifiers
- Fumbles ignored on the second attack roll

## Katas (Multiple Attacks)

Making multiple attacks in a single action.

### Requirements
- Each attack must use a different limb (one weapon per hand, kick with leg, etc.)
- Off-hand weapon: -20 penalty (unless Ambidextrous talent)

### Penalties

| # Attacks | Modifier |
|-----------|----------|
| 2 | -75 each |
| 3 | -100 each |
| 4 | -125 each |
| 5 | -150 each |
| Per additional foe | -20 each |
| Primary weapon is light | +10 |
| Off-hand is dagger/natural | +20 |
| Off-hand is other light weapon | +10 |

- Combat Expertise: Multiple Attacks skill offsets these penalties
- Light weapon bonuses cannot create a net bonus, only reduce penalties

## Protecting Others

A combatant can use parry or shield to protect an ally.

### Requirements
- Protector must be within melee range of both combatants
- Protector takes OB penalty based on position
- Protector designates how much of his parry/shield DB protects the ally instead of himself
- Can wait until attack is declared before deciding

### Protect Penalties

| Scenario | Modifier |
|----------|----------|
| Protector between Attacker and Defender | -25 |
| All other scenarios | -50 |
| Defender between Attacker and Protector | -100 |

- Battle Expertise: Protect skill offsets these penalties
- Grond's Protect skill: +36

## Slaying Attacks

Some attacks (often magical) are exceptionally dangerous. Slaying adds directly to the critical roll.

| Type | Critical Roll Bonus |
|------|-------------------|
| I | +5 |
| II | +10 |
| III | +15 |
| IV | +20 |
| V | +25 |

## Subdual (Non-Lethal)

Attacks intended to knock out or subdue without killing.

- Subdual attacks do **half** concussion hit damage
- Use the **Subdual critical table** instead of normal critical type (max severity E)
- Combat Expertise: Subdual skill offsets penalties

### Subdual Penalties

| Attack Type | Modifier |
|-------------|----------|
| Unarmed, blackjack, sap | -25 |
| Crushing (mace, club, ice bolt) | -50 |
| Blades (flat side), claws | -75 |
| Piercing (bite, stinger) | -100 |
| Ranged* | -50 |
| Unfamiliar anatomy* | -50 |

*Cumulative with other penalties

---

## Touch Attacks (Spell Delivery)

- Use any unarmed combat skill to deliver a touch spell in melee
- Any result of 1 hit or more = successful spell delivery
- Can declare "tag only" (no damage) before rolling
- Friendly targets don't actively defend, but still get Quickness DB, magical armor, etc.
- Foe who declared parry gets full parry DB
