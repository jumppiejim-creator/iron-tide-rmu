# Treasure Generation Rules Reference

## Overview

NPC and creature treasure is determined by three codes: Wealth (coins, gems, jewelry), Lifestyle (gear quality), and Magic (magical items). Each is rated from Very Poor to Very Rich.

It is best to determine treasure in advance. Intelligent beings will generally use the best items they have — a bandit with a magic mace in his pack but fighting with a club makes no sense.

## Lifestyle / Gear Quality (Table 7-1f)

The Lifestyle code indicates overall gear quality. Roll d100 and cross-reference with the creature's lifestyle code. The result is a total bonus to distribute across gear items. Where two numbers are listed (e.g., +20/+10), the second is the maximum for any single item.

Leaders of large groups may roll as one wealth rating higher.

| Roll | Very Poor (l) | Poor (m) | Normal (n) | Rich (o) | Very Rich (p) |
|------|---------------|----------|------------|----------|---------------|
| 01-05 | -15 | -10 | -5 | 0 | 0 |
| 06-20 | -10 | -5 | 0 | +5 | +10 |
| 21-50 | -5 | 0 | 0 | +10 | +15/+10 |
| 51-80 | 0 | 0 | +5 | +15/+10 | +20/+10 |
| 81-95 | 0 | +5 | +10 | +25/+10 | +30/+15 |
| 96-100 | +5 | +10/+5 | +15 | +30/+15 | +40/+20 |

## Magic Items (Table 7-1g)

Roll d100 and cross-reference with the creature's magic code. Results indicate modifiers for rolls on the item type and specific item tables. If the modified result is less than 1, no item is found. Multiple modifiers mean multiple possible items — roll each separately.

| Roll | Very Poor (u) | Poor (w) | Normal (x) | Rich (y) | Very Rich (z) |
|------|---------------|----------|------------|----------|---------------|
| 01-20 | none | none | none | none | none |
| 21-40 | none | none | none | none | -50 |
| 41-55 | none | none | none | none | +50 |
| 56-70 | none | none | -50 | +0 | +70, -50 |
| 71-80 | -80 | -60 | +0 | +20, -50 | +80, +0 |
| 81-90 | -70 | -50 | +10, -50 | +40, -40 | +100, +0 |
| 91-94 | -60 | -40 | +20, -40 | +60, -30 | +120, +20, -80 |
| 95-97 | -50 | -30 | +30, -30 | +80, -20 | +150, +30, -70 |
| 98-99 | -40 | -20 | +40, -20 | +100, -10 | +180, +40, -60 |
| 100 | -30 | +0 | +50, -10 | +120, +0, -50 | +200, +50, -50 |

**Max Item Level:** Very Poor 10, Poor 20, Normal 35, Rich 50, Very Rich no limit.

### Table 7-1h: Item Type

For each modifier from Table 7-1g, roll d100 to determine the category, then roll on the appropriate sub-table adding the modifier.

| Roll | Item Type | Sub-Table |
|------|-----------|-----------|
| 1-20 | Melee Weapons and Missiles | Table 7-1i |
| 21-35 | Ranged Weapons | Table 7-1l |
| 36-50 | Armor | Table 7-1m |
| 51-70 | Utility Items | Table 7-1o |
| 71-85 | Spell Items | Table 7-1p |
| 86-100 | Charged and Casting Items | Table 7-1q |

*Note: The full sub-tables (7-1i through 7-1q) contain hundreds of specific items and are too large for this reference. For the complete tables, consult Treasure Law Chapter 7 directly. Common results at low modifiers include: +5/+10/+15 weapons, +5/+10 armor, minor PP adders, casting focuses, and low-level potions.*
